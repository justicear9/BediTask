Installation

Quick Installation Guide
  • Go to Settings > Plugins.
  • Install the Login-as-Client-plugin-for-RISE-CRM.zip which you have downloaded from the Envato Market.
    It'll require the item purchase code. Don't use any nulled version for your security.
  • If everything is OK, it'll show a success message.
  • Activate the plugin from the option dropdown.

Features

  • Once you login as an admin, you'll see an additional icon in your topbar.


  • After clicking on that button, there'll be a modal containing client contacts list.


  • After logging as a client, you'll get a button to simply get back to your own account.

Upgrading Guide

Please note that you can only download the updates if you had a valid purchase code on installation.

Please backup all files and database before start the installation of updates.

There is a very easy way to update your application.

To install any updates go to Settings > Plugins > Login as Client > Dropdown > Updates. The download will be start automatically once any new update is available. Once the download is completed you can install the updates by clicking on the link. That’s all!
